# ============================================================================
# Installation Notes
# ============================================================================
#
# IMPORTANT: This project requires Python 3.14.0 and NVIDIA GPU with CUDA 12.x
#
# Hardware Requirements:
#   - CPU: AMD Ryzen 7 7700X (or equivalent)
#   - GPU: NVIDIA RTX 5090 with 24GB VRAM (or equivalent RTX 40/50 series)
#   - RAM: 32GB DDR5
#   - CUDA: 12.x with cuDNN 8.9+
#
# System Dependencies:
#
# 1. NVIDIA Drivers & CUDA Toolkit:
#    - Install NVIDIA Driver 550+ (for RTX 5090)
#    - Install CUDA Toolkit 12.6+: https://developer.nvidia.com/cuda-downloads
#    - Install cuDNN 8.9+: https://developer.nvidia.com/cudnn
#    - Verify: nvidia-smi && nvcc --version
#
# 2. TA-Lib (Technical Analysis):
#    - Ubuntu/Debian: apt-get install ta-lib
#    - macOS: brew install ta-lib
#    - Windows: Download from https://www.lfd.uci.edu/~gohlke/pythonlibs/#ta-lib
#
# 3. QuantLib:
#    - Ubuntu/Debian: apt-get install libquantlib0-dev
#    - macOS: brew install quantlib
#    - Windows: Requires manual compilation or pre-built binaries
#
# 4. PostgreSQL:
#    - Ubuntu/Debian: apt-get install postgresql-server-dev-all
#    - macOS: brew install postgresql
#
# 5. Redis:
#    - Ubuntu/Debian: apt-get install redis-server
#    - macOS: brew install redis
#
# Installation Steps:
#
# Step 1: Install system dependencies (CUDA, cuDNN, etc.)
#   sudo apt update
#   sudo apt install nvidia-driver-550 cuda-toolkit-12-6
#   # Follow NVIDIA instructions for cuDNN
#
# Step 2: Install PyTorch with CUDA support (CRITICAL):
#   pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
#
# Step 3: Install TensorFlow with GPU support:
#   pip install tensorflow[and-cuda]==2.18.0
#
# Step 4: Install remaining dependencies:
#   pip install -r requirements.txt
#
# Step 5: Verify GPU installation:
#   python -c "import torch; print(f'CUDA available: {torch.cuda.is_available()}')"
#   python -c "import tensorflow as tf; print(f'GPUs: {tf.config.list_physical_devices(\"GPU\")}')"
#
# For development:
#   pip install -r requirements.txt
#   pip install -r requirements-dev.txt  # If you split dev dependencies
#
# Hugging Face Models:
#   # Models will be automatically downloaded on first use
#   # Recommended models for fine-tuning:
#   #   - ProsusAI/finbert (financial sentiment)
#   #   - AutonLab/MOMENT-1-large (time series)
#   #   - google/timesfm-1.0-200m (forecasting)
